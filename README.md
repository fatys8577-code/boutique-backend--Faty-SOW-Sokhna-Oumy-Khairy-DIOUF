# Gestion de boutique - Backend Laravel

## Présentation du projet

Ce projet est une application backend de gestion de boutique développée avec Laravel.

L'application permet de gérer :

- Les catégories de produits
- Les produits
- Les acheteurs
- Les achats

Elle possède également une API REST documentée avec Swagger et une gestion des utilisateurs avec différents rôles.

---

## Technologies utilisées

- Laravel
- PHP 8.3
- MySQL
- Blade
- API REST
- Swagger (L5-Swagger)
- Composer
